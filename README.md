# Bugatti Veyron — local mini website (EN)

## Open the site
1. Unzip the folder (if needed).
2. Double‑click `index.html` (or right‑click > Open with > your browser).

## Share
- Zip the whole folder and send it.
- The site is 100% static and works offline.

## Replace the illustration with a photo
- Put your image in `assets/` (e.g., `assets/veyron.jpg`)
- In `index.html`, replace:
  `assets/veyron-illustration.svg` with `assets/veyron.jpg`

## Generate a PDF
- Open `technical.html`
- Click “Print / Save as PDF”
